# Mars zsh theme

## Requirements
- [zsh shell](https://www.zsh.org)
- [Oh My Zsh](https://ohmyz.sh)

## Instalation
Download theme file

```
curl https://raw.githubusercontent.com/aorek/mars-zsh-theme/main/mars.zsh-theme --output $ZSH_CUSTOM/theme/mars.zsh-theme
```
